# ETC

## 모듈?

> `import module`

* 장점
  * 유지보수 비용(휴먼리소스) 감소

* 단점
  * 모듈.메서드 - DRY원칙 모듈 반복도 원칙에 어긋남

> `import from module import` 메서드

* 장점
  * 모듈 안에 있는 메서드를 사용하기 위한 타이핑 줄어듬
  * 모듈에서 특정 부분만 해당 스크립트에서 접근 가능하도록 제어

* 단점
  * 모듈에서 새로운 항목 추가하려면 import 구문 업데이트 해야함
  * random이라는 모듈에서 `shuffle()` VS `random.shuffle()`
    * 뒤에것이 어디에서 가져왔는지 보임
* 어떤게 좋은지는 맘대로
